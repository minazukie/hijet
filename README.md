# HiJet

Start a suitable JetBrains IDE with one command.

## Requirements

- Platform: OSX only.
- Python 3.6 and up.
- Make sure you have installed a full JetBrains suite.

## Installation

```shell script
pip3 install git+https://github.com/minazukie/hijet
```

## Usage

```shell script
cd your-project
hijet
```

## Flags

- -e exclude directory
- -d specify project directory
